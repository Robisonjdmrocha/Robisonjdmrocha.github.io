# Portfólio — Robison Rocha (GitHub Pages)

Este repositório contém um portfólio simples e objetivo, otimizado para recrutadores.

## Como publicar (GitHub Pages)

1. Crie um repositório chamado **Robisonjdmrocha.github.io**
2. Faça upload dos arquivos deste projeto (ou clone e dê push)
3. Vá em **Settings → Pages**
4. Em **Build and deployment**, selecione:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Acesse: https://robisonjdmrocha.github.io

## Como editar

- Conteúdo: `index.html`
- Estilo: `css/style.css`
- Script: `js/script.js`
